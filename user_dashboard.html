How to Use This Tool

1. Use the map to identify hazard locations or click "Use My Current Location" to geotag your report

2. Select the hazard type and severity level

3. Add a description of the hazard and upload photos/videos as evidence

4. Click "Add Hazard Report" to submit your report to the map

5. Click on any hazard marker to view details and media
